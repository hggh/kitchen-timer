# kitchen-timer


## parts

 * Atmel 328p (1Mhz, BOD disabled)
 * 32 khz crystal (timer2, async)
 * TM1637 – 4 Digit 7 Segment Display
 * Rotary encoder KY-040
 * TP4056 with DW01-P
 * LiPo 18650
 * buzzer
